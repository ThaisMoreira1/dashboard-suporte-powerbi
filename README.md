# 📊 Dashboard de Suporte Técnico

Projeto desenvolvido para análise de indicadores de suporte técnico, com foco em eficiência operacional e experiência do cliente.

---

## 🔍 Indicadores analisados:
- Volume de chamados
- Tempo médio de atendimento
- SLA (nível de serviço)
- Chamados por prioridade
- Satisfação do cliente

---

## 🛠️ Ferramentas utilizadas:
- Power BI
- Excel

---

## 🎯 Objetivo:
Auxiliar na tomada de decisão através de dados, identificando gargalos no atendimento e oportunidades de melhoria nos processos de suporte.

---

## 💡 Sobre o projeto:
Este projeto simula um ambiente real de suporte técnico, onde dados são analisados para melhorar a performance do atendimento e a experiência do cliente.
